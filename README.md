# Sprint 5 - Автотесты для Stellar Burgers

Тесты для учебного приложения космического фастфуда.

## Запуск тестов
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите тесты: `pytest tests/ -v`

## Тестовое покрытие
- Регистрация пользователя
- Вход в аккаунт
- Личный кабинет
- Конструктор бургеров
